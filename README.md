# pokemon-live
A real time geolocation pokemon map for Pokemon 


# Getting start
1. Install Meteor:
   Linux: curl https://install.meteor.com/ | sh
   Windows: [download](https://www.meteor.com/install)
   
2. Navigate to the project folder:
   `$meteor`
