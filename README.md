# pokemon-live
A real time geolocation pokemon map for Pokemon Go.


# Getting start
1. Install Meteor:
   Linux: `curl https://install.meteor.com/ | sh`
   Windows: [download](https://www.meteor.com/install)
   
2. Navigate to the project folder:
   `$meteor`
   Now your should see  
   `App running at: http://localhost:3000/`
